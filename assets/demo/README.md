# assets/demo

데모 영상과 스크린샷을 두는 곳.

## 파일 목록 (갱신 예정)

- `weather-menu.mp4` — `examples/weather-menu`를 렌더한 결과물 (1080×1920, 10초)
- `studio-timeline.png` — Remotion Studio의 타임라인 스크린샷
- `render-output.png` — 최종 렌더 결과 썸네일

## 생성 방법

```bash
cd examples/weather-menu
npm install
npm run build
cp out/weather-menu.mp4 ../../assets/demo/
```

용량이 2MB를 넘으면 Git LFS 사용을 고려할 것.
